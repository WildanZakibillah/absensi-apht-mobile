# 📖 Tentang Project

**AbsenKu** merupakan aplikasi absensi mobile yang dikembangkan untuk mendukung sistem kehadiran digital di lingkungan **APHT Sumenep**.

Aplikasi ini dibuat menggunakan **Kotlin** di **Android Studio** dengan integrasi **Firebase** sebagai backend service.

Fokus utama pengembangan aplikasi ini adalah:

* Kemudahan penggunaan
* Tampilan modern & clean UI
* Proses absensi realtime
* Efisiensi pencatatan kehadiran
* Pengelolaan data yang lebih praktis

# 🚀 Cara Menjalankan Project

## 1️⃣ Clone Repository

```bash
git clone https://github.com/USERNAME/absenku.git
```

---

## 2️⃣ Buka Project di Android Studio

* Open Android Studio
* Klik `Open Project`
* Pilih folder project `AbsenKu`

---

## 3️⃣ Sync Gradle

Tunggu hingga proses:

* Gradle Sync
* Dependency Download
* Build Project

selesai.

---

## 4️⃣ Setup Firebase

Tambahkan file:

```bash
google-services.json
```

ke folder:

```bash
app/
```

---

## 5️⃣ Jalankan Aplikasi

* Hubungkan device Android
  ATAU
* Gunakan Android Emulator

Klik tombol:

```bash
▶ Run
```
